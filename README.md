# Phoenix Hub

## What is Phoenix Hub?

Phoenix Hub is a platform where student organizations can sell their Merchandise to students.

## Technologies Used

* CodeIgniter4 
* Shield
* Paymongo
* Bootstrap
* Ajax
* Jquery
* MySQL
* Cloudinary
